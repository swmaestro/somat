# 데이터

### 데이터 추가하기
---
* 데이터는 anamlife.json에 저장됩니다.
* data array의 마지막에 데이터를 추가해주세요.
	* **추가하시기 전에 콤마를 꼭 확인해주세요!**
* data는 다음 형식으로 구성되어 있습니다.
	* name : 업체명
	* content : 간단한 소개
	* x : 경도
	* y : 위도
	* tag : 태그 (태그 목록에서 해당되는 태그를 추가해주시면 됩니다.)

### 팁
---
* 좌표는 네이버 / 다음 지도에서 업체명을 검색하신 다음 주소를 구글 지도에서 검색하시면 편하게 추가하실 수 있습니다.
